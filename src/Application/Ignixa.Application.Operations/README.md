# Ignixa.Application.Operations

This library contains the core FHIR operation implementations for the Ignixa FHIR Server.

## Description

It includes handlers for standard and extended FHIR operations such as:
- `$validate`
- `$everything`
- `$expand`
- `$lookup`

**Note:** This is an internal component of the Ignixa FHIR Server and is not intended to be used directly by external applications.
