# Ignixa.Application.BackgroundOperations

This library contains background operation handlers for the Ignixa FHIR Server.

## Description

It includes implementation for long-running tasks such as:
- `$export` (Bulk Data Export)
- `$import` (Bulk Data Import)
- Background job processing infrastructure

**Note:** This is an internal component of the Ignixa FHIR Server and is not intended to be used directly by external applications.
