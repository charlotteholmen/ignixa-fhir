# Ignixa.DataLayer.BlobStorage

This library provides blob storage integration for the Ignixa FHIR Server.

## Description

It handles the storage of large binary data (attachments, bulk export files) using blob storage services (e.g., Azure Blob Storage, AWS S3) or local emulation.

**Note:** This is an internal component of the Ignixa FHIR Server and is not intended to be used directly by external applications.
