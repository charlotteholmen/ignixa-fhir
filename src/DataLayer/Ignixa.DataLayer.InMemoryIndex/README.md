# Ignixa.DataLayer.InMemoryIndex

This library provides an in-memory search index for the Ignixa FHIR Server.

## Description

It implements high-performance search capabilities for the File System data provider, allowing for:
- Fast resource lookups
- Parameter indexing
- In-memory query execution

**Note:** This is an internal component of the Ignixa FHIR Server and is not intended to be used directly by external applications.
